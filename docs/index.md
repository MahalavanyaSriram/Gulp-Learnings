# Front End tutorial

This consist of various samples of most of the topics in HTML5, CSS ,BOOTSTRAP,JS,ES6 and ANGULAR4. 


## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Node.js and NPM 

### Installing

Intall node.js which is an open source form nodejs.org and npm from npm.org

setup your environment accordingly

Install the below mentioned dependencies from node cmd using npm

Install gulp using npm install gulp -g

Install gulp-util using $ npm install --save-dev gulp-util

Install gulp-sass using $ npm install --save-dev gulp-sass

Install
